Core Insight: This paper solves the problem of temporal inconsistencies in generated traffic videos, which often exhibit flickering and unrealistic motion trajectories. The key insight is the introduction of Localized Semantic Alignment (LSA), which aligns semantic features between ground-truth and generated video clips, focusing on dynamic objects to enhance coherence across frames.

Key Contribution: LSA employs three main components: (1) It extracts features from both ground-truth and generated frames using DINOv2, producing feature maps of size H/p × W/p × d, where p is the patch size and d is the feature dimension. (2) It computes a semantic feature consistency loss, defined as L_feat = E[∥(f(i,j)−f(i,j))⊙m(i,j)∥²], which emphasizes dynamic object regions using masks. (3) It fine-tunes the Stable Video Diffusion model by combining this loss with the standard diffusion loss, optimizing the model to learn consistent motion patterns within a single epoch.

Result & Impact: On the nuScenes dataset, LSA fine-tuned models achieve a +5.2% improvement in mAP and mIoU metrics compared to the baseline SVD model. The qualitative results show reduced flickering and improved object motion consistency, demonstrating the effectiveness of LSA in enhancing temporal coherence in generated videos.

Technical Essence: By applying localized semantic alignment through dynamic object masks during training, LSA significantly reduces temporal inconsistencies in generated videos, leading to improved alignment of object motion and appearance without requiring additional inference-time complexity.

Technical Essence: By employing a calibration-free implicit BEV projection that utilizes cross-attention between image patches and BEV queries, ViGT achieves a unified scene representation that enhances occupancy estimation accuracy across diverse sensor configurations.
