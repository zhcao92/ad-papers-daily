arXiv: 2601.09812v1 | PDF | Published: 2026-01-14

Core Insight: This paper solves the problem of high false positives and missed detections in 3D object detection by fusing LiDAR and RGB data effectively. The key insight is to use a late-cascade fusion approach that matches 3D LiDAR detections with 2D RGB detections, filtering out unmatched LiDAR detections and recovering missed objects from unmatched RGB detections.

Key Contribution: LCF3D employs three key modules: (1) a Bounding Box Matching module that projects 3D LiDAR boxes onto the image plane and matches them with 2D detections using IoU optimization, filtering out unmatched LiDAR detections; (2) a Detection Recovery module that backprojects unmatched 2D boxes into 3D frustums, applying a 3D localization model to recover missed objects; (3) a Semantic Fusion module that resolves label inconsistencies by favoring RGB predictions. This structure allows parallel processing of RGB and LiDAR data, reducing computational overhead.

Result & Impact: On the KITTI dataset, LCF3D achieves a 5.2% improvement in mAP for pedestrians and cyclists compared to traditional LiDAR-based methods. In nuScenes, it shows a 4.1% increase in mAPH for bicycles and motorcycles. The framework runs at 15 FPS, significantly faster than the 8 FPS baseline of early fusion methods, making it suitable for real-time applications.

Technical Essence: By implementing a Bounding Box Matching module that filters LiDAR false positives and a Detection Recovery module that backprojects unmatched 2D detections into 3D frustums, LCF3D recovers missed detections, achieving a 5.2% increase in mAP on KITTI.

Core Insight: This paper solves the problem of feature redundancy in patch descriptors from foundation models, which can lead to overfitting and inefficiencies in autonomous driving systems. Key Contribution: The authors introduce Stochastic Patch Selection (SPS), which operates in two main steps: (1) For each input frame, a random subset of patch descriptors is selected based on a keep-rate (e.g., RATE = 0.4), effectively reducing the number of descriptors fed into the policy network. (2) The remaining descriptors maintain their spatial layout, allowing the policy to learn from coherent yet diverse views of the scene. This process reduces computational load by approximately 2.5× while enhancing out-of-distribution (OOD) generalization.

Result & Impact: On closed-loop driving simulations, SPS achieves an average improvement of +6.2% in driving success across various OOD scenarios and up to +20.4% in specific tests. The method also reduces the computational latency from 200ms to approximately 80ms, significantly enhancing real-time performance in autonomous driving applications.

Technical Essence: By implementing Stochastic Patch Selection to randomly mask a fraction of patch descriptors, the system reduces reliance on redundant features, leading to a 2.4× increase in processing speed and a 6.2% improvement in OOD generalization.
